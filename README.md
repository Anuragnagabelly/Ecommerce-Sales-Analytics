📌 Project Overview
This project analyzes E-commerce sales data to extract business insights and perform customer segmentation using RFM analysis.

🛠 Tools Used

Python (Pandas, Matplotlib)

SQL Server (CTE, Window Functions)

Power BI (DAX, Drill-through, Dashboard Design)

2018 recorded highest revenue

Others segment contributes 53% revenue

Champions contribute 21.6% revenue

Repeat rate only 2.97%

Late delivery rate 6.5%
